# Personal Website — Starter

This is a minimal static website starter.

Included files:
- `index.html` — home page
- `styles.css` — styling
- `favicon.ico` — optional (add your icon)

How to use:
1. Edit `index.html` to add your name, bio, projects, and contact info.
2. Commit and push these files to your repository (e.g., the root of the `main` branch).
3. Enable GitHub Pages:
   - Go to your repository Settings → Pages.
   - Choose the branch (`main`) and folder (`/ (root)`), then Save.
   - Your site will be published at `https://<your-username>.github.io/<repo>` (or at your custom domain if configured).
